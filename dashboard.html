Having issues exporting video? Click here! 

When you select export video, the software will record the screen. Once it is complete, you will receive a .webm file to your download folder.

You can preview the .webm file by dragging it into your browser.

To convert .webm to mp4 (for posting on social media etc), you can use a free online tool. Here are a couple that we suggest: 
1) https://www.freeconvert.com/webm-to-mp4/download
2) https://cloudconvert.com/webm-to-mp4

To get best results, we recommend using google chrome on desktop or laptop. Tablet video export is not yet supported.

If you still cannot view the video, we suggest recording the screen using quicktime (mac), or the equivalent PC program. You can do the same on tablet.
